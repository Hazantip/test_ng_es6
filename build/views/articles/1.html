Назад в будущее
Вернёмся к нашему примеру с ES6, который я, к сожалению, не могу запустить на своём браузере образца апреля 2015 года. Может ли нам помочь webpack? Легко! В webpack есть понятие загрузчиков, дополнительных модулей, которые добавляются в конфигурацию, чтобы загружать файлы, соответствующие какому-то признаку. Есть целая огромная куча загрузчиков для самых различных вещей, не только для JavaScript, а даже для CSS или изображений.
